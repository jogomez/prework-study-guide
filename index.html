HTML

  • The header element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element appears in the browser's title bar.
  • The section elements organize the content based on topics or categories.

CSS

  • CSS is a style sheet language
  • The whole CSS structure is called a ruleset
  • CSS Selector is the HTML element name at the start of the ruleset.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

GIT

  • Verify working branch is Main (git status) - If you need to come back to main ise (git checkout main)
  • Create a new branch and move the working branch to the new one (git checkout -b feature/starter-code)
  • Verify the new branch is created (git status)
  • Add the starter code to the staging area of the current working branch (git add -A)
  • Commit the changes in staging area (git commit -m "add starter code”)
  • Verify there are no new changes in main, i.e. the new branch and main are in-sync (git pull origin main)
  • Push the changes to the remote branch (git push origin feature/starter-code)
  • Go to the repo in the browser, select the new branch, and select “Compare and pull request”
  • Add a title in the Pull Reqest (PR), select base branch = main and comparison branch as the new branch (feature/starter-code)
  • Select “Create Pull Request”
  • People will review and comment on the PR, otherwise select “Merge pull request”

JavaScript

  • JavaScript is the world's most popular programming language.
  • JavaScript is the programming language of the Web.
  • JavaScript is easy to learn.
  • A variable is a named container that allows us to store data for our program.
  • Control flow is the order in which a computer executes code in a script.
